# Decision Trees Project - Fundamentals of AI

## Goals
- [ ] **Report (Реферат):** Comprehensive theoretical and practical overview of Decision Trees.
- [ ] **Presentation (Презентация):** Slides summarizing the key concepts, algorithms, and results.
- [ ] **Example Code (Примерен код):** Python/MATLAB implementation of Decision Trees on provided/standard datasets (Iris, Diabetes).

## Context from Lab6_cdt.pdf
- Topic: Classification with Decision Trees.
- Dataset: `fisheriris` (Iris dataset).
- Key Tasks:
    - Binary classification (one class vs others).
    - Feature importance (informativeness).
    - Accuracy measurement via Confusion Matrix.
    - Application to Diabetes dataset.

## Technology Stack (Approved)
- **Report:** Word Document (`.docx`) generated via `python-docx`.
- **Presentation:** PowerPoint (`.pptx`) generated via `python-pptx`.
- **Code:** Python (`scikit-learn`, `pandas`, `matplotlib`, `graphviz` optional).

## Working Directory Structure
- `/report`: Contains the generated `.docx` report.
- `/presentation`: Contains the generated `.pptx` presentation.
- `/code`: Implementation examples (`decision_tree_example.py`) and generation scripts.
